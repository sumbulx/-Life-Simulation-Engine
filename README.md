# Life Simulation Engine (Text-Based Simulator)

## Overview

Life Simulation Engine is a C++ console-based simulation project developed using Object-Oriented Programming principles. The application allows users to control a virtual character and make everyday life decisions that influence future outcomes. Through activities such as studying, working, exercising, and resting, the simulator demonstrates how consistent choices can impact personal growth, financial stability, health, and overall happiness.

## Features

* Interactive text-based simulation
* Realistic decision-making system
* Dynamic character progression
* Multiple life attributes:

  * Health
  * Energy
  * Money
  * Skill
  * Happiness
* Random events and challenges
* End-of-simulation performance analysis
* Structured and modular OOP architecture

## Technologies Used

* C++

* Console-Based Application Development

## OOP Concepts Applied

* Classes and Objects
* Encapsulation
* Inheritance

* Polymorphism
* Modularity

## How the Simulation Works

The game progresses through a series of virtual days where the player selects actions that directly affect the character's attributes. Every choice carries advantages and trade-offs, encouraging strategic thinking and long-term planning. Random life events further enhance realism by introducing unexpected opportunities and obstacles. At the conclusion of the simulation, the system generates a comprehensive report evaluating the player's decisions and overall life balance.
